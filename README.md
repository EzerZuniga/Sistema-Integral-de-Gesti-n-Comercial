# Sistema-Integral-de-Gesti-n-Comercial
Sistema Integral de Gestión Comercial es una plataforma desarrollada para optimizar y centralizar las operaciones de una bodega minorista. Su objetivo principal es ofrecer un entorno intuitivo, seguro y eficiente para la administración de inventarios, procesos de compra y venta, control de stock, así como la gestión de proveedores, trabajadores.
